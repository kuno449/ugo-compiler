About UgoCompiler

UgoCompiler is a free online Java compiler.
It’s made as a sub project of the blog Ugo.

UgoCompiler
  • is free
  • don’t need registration
  • uses API of Wandbox (https://wandbox.org, https://github.com/melpon/wandbox)
